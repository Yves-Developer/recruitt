# Welcome to Recruitt Documentation

Recruitt is a modern platform managed via a Turborepo monorepo.

## Project Structure

- `apps/web`: Next.js Frontend
- `apps/server`: Express.js Backend
- `apps/docs`: Docus Documentation
- `packages/shared`: Shared types and schemas
- `packages/ui`: Shared UI components

## Quick Start

```bash
pnpm install
pnpm dev
```
